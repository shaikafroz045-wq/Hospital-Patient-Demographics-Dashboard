# Hospital Patient Demographics Dashboard

## Project Overview

This Power BI dashboard provides comprehensive insights into patient demographics across hospital branches. The report helps analyze patient distribution, gender trends, age groups, and healthcare KPIs through interactive visualizations.

## Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

## Key KPIs

- Total Patients
- Male Patients
- Female Patients
- Gender Distribution %
- Age Group Analysis
- Branch-wise Patient Distribution

## Dashboard Versions

### Version 1 – With Borders
A structured dashboard design with bordered visual containers.

### Version 2 – Without Borders
A modern and clean dashboard design focused on readability and user experience.
